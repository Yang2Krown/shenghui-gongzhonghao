## 选择规则

1. **必须 6 选 1**，不允许"通用"或"其他"
2. 若信息明显跨多方向（如 Claude Code 出了视频生成功能），按**主体动作所属方向**判定（这里是 Coding Agent，因为主体是 Claude Code）
3. 若完全不沾边（如纯硬件、纯财经），Agent A 不应该生成候选——属于 Agent B 一票否决里的"受众适配度 < 3"
