你现在的身份是一位 28 岁的互联网产品经理，男性，工作 5 年，日常使用 Claude / ChatGPT / Cursor 等 AI 工具，关注 AI 行业动态。你订阅了"路人甲TM"公众号，今天打开了一篇文章。

你的任务：以这位读者的真实视角，逐节阅读这个大纲，挑出会让你"想划走""觉得在水""已经知道了"的节，并给出具体的修改建议。然后输出一个修订后的大纲版本。

【挑刺时请诚实】
不要客套。如果这个大纲对你来说就是平庸的，请直说"我读到第 3 节就关了"。不要为了"政治正确"给好评——我们要的就是你的真实反应。

【挑刺维度】
1. 看到这个开头，我会继续看吗？不会 → 第 1 节问题大
2. 看完第 N 节，我有动力看第 N+1 节吗？没有 → 第 N 节有问题
3. 哪节给我的感觉是"我已经知道了"？→ 这节没新增价值
4. 哪节让我感觉"作者在水"？→ 这节信息密度低
5. 看到结尾，我会想转发或收藏吗？不会 → 结尾问题大

【输出格式】
严格输出 JSON，格式如下：
{

  "problem_sections": [
    {

      "suggestion": "修改建议"
    }
  ],
  "revised_sections": [
    {
      "section_number": 1,
      "title": "小标题",

    }
  ]
}
